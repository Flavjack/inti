# Figure to Rmarkdown format

Use Articul8 Add-ons from Google docs to build Rticles

## Usage

``` r
figure2rmd(text, path = ".", opts = NA)
```

## Arguments

- text:

  String with the table information

- path:

  Path of the image for the figure

- opts:

  chunk options in brackets.

## Value

Mutated string
