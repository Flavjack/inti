# Table to Quarto format

Use Articul8 Add-ons from Google docs to build Rticles

## Usage

``` r
table2qmd(text, type = "asis")
```

## Arguments

- text:

  Markdown text with table information (string)

- type:

  output file type `[chracter: "asis" "list", "listfull", "full"]`.

## Value

string mutated
